## Python Programming Projects

This repository contains a collection of Python programs developed
for academic practice and learning purposes.

The projects include:
- Ticket issuing systems for train and plane transportation
- Graphical user interface (GUI) applications built using Tkinter

These programs demonstrate basic programming logic, user input
handling, conditional statements, and simple GUI design.

## Files
- train_ticket_issuer.py – Train ticket issuing system
- plane_ticket_issuer.py – Plane ticket issuing system
- calculator_gui.py – Calculator application with GUI
- temperature_converter_gui.py – Temperature converter with GUI

## Tools Used
- Python
- Tkinter
